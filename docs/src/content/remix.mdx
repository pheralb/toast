# Usage with Remix

How to use the library with [Remix **v2** with Vite](https://remix.run/).

1. Create a new Remix project:

```bash
npx create-remix@latest
```

2. Select the default options:

3. Install the library:

```bash
npm install @pheralb/toast
```
